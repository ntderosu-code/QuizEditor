Interoperability, not lock-in

Author once. Open it wherever you teach.

QTI and IMS Common Cartridge are open standards, not one vendor's feature. Quiz Editor reads and writes them faithfully, so a quiz you write today still imports cleanly into next year's platform.

QTI 1.2 QTI 2.1 .imscc .quizeditor

Imports and exports: Canvas, Brightspace, Blackboard, Moodle, D2L, and any system that speaks QTI or Common Cartridge.

Every QTI export is checked before it is written: well-formed XML, a consistent manifest, and a clean re-import that yields the same questions. Problems surface in the app, not after you upload.

Authoring

Every question type, written like prose.

Multiple choice, multiple answer, true/false, fill-in-the-blank, short answer, essay, and matching. Write prompts and feedback as real rich text, with images, tables, and links that survive the round trip through QTI. Images cannot leave the app without alt text.

  • Tag, rate, and weight every question with topics, difficulty, and points.
  • Find anything fast with live sidebar search, tag and difficulty filters, and a quick-switch palette.
  • Reorder by dragging, duplicate in a keystroke, and undo any structural change.

Review & AI

A reviewer that reads for item-writing, not just spelling.

An offline linter flags the classics instantly, no network required: no correct answer, an unemphasized negative stem, a giveaway longest option, all-of-the-above, missing feedback. When you want more, AI review returns a before-and-after you apply field by field, and can draft new questions, distractors, and feedback.

Bring an OpenAI-compatible API, run Apple's on-device models, or copy a prompt into any assistant and paste the reply back. Results open in a window you can read, copy, or save.

Import & export

Bring questions in. Send finished exams out.

Import a QTI .zip, an IMS Common Cartridge, or loosely marked plain text. Pull in exactly the questions you want, from a whole course cartridge or a single item bank, and merge banks across files.

  • A question picker with search, select-all, and per-question preview before anything lands.
  • A read-only question bank across a folder of saved quizzes.
  • Export validated QTI, a formatted document, or a print-ready paper exam with a separate answer key.

Accessibility

Accessible is the floor, not a feature.

WCAG 2.1 AA is the baseline this app is built to, and it ships that way the first time, not as a follow-up.

  • Alt text is required on every image before export, or an explicit decorative choice.
  • Full keyboard operation with a visible focus ring on every control.
  • VoiceOver labels throughout, and status changes announced as they happen.
  • Color is never the only signal. A correct answer carries a mark and a label, not just a hue.
  • Dynamic Type is respected, so the interface scales with the reader.

Native and local

Yours to keep, on a Mac that already trusts it.

Built entirely on Apple frameworks, SwiftUI, AppKit, and WebKit, with no third-party dependencies. Quizzes are plain-JSON .quizeditor documents you can diff, back up, and own. Open, edit, review, and export entirely offline. MIT licensed.

Get started

Clone it and run.

It is open source. Build it from the repository with a Swift 6 toolchain on macOS 14 or later. The full test suite runs with one command.

Open the repository

# clone, then run the app
$ git clone https://github.com/ntderosu-code/QuizEditor.git
$ cd QuizEditor
$ swift run QuizEditorApp

# run the tests
$ swift test
